# Aryan Singh | 3D Cinematic Portfolio

A high-performance, cinematic 3D portfolio showcasing the dual-threat expertise of **Aryan Singh**—Software Engineer and Film Director.

## ✦ Key Features

- **Cinematic 3D Experience:** Atmospheric backgrounds and fluid animations using Framer Motion and custom canvas particles.
- **AI Assistant:** Integrated natural language processing for interactive, voice-controlled navigation and Q&A.
- **Responsive Scrollytelling:** Smooth scrolling architecture for a studio-quality UX.
- **Tech Stack:** React 19, Vite, Tailwind CSS, Framer Motion, and Web Speech API.

## ✦ Run Locally

**Prerequisites:** Node.js

1. **Clone the repository**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```

## ✦ Deploy

This project is optimized for deployment on **Vercel** or **Netlify**. Simply connect your GitHub repository and the build settings will be automatically detected.
